# ¿Cómo conectar el DASHBOARD con la base de datos?
Antes que nada, ejecutar la query "dashboard user.sql" ya que el dashboard usa su propio usuario con permisos de solo lectura.

Para conectar con tu servidor de DB ir a Inicio -> Transformar Datos -> Editar Parametros
Remplazar en "Server" por la dirección de tu servidor local SQL y en "Database" dejar VibeFit a no
ser que usaras otro nombre. Pero no creo.
